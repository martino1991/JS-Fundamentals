  1. HTML

    2. The HTML language describes Web content (Web pages)

    Text with formatting, images, lists, hyperlinks, tables, forms, etc.

    Uses tags to define elements in the Web page

    HTML Elements : Има два вида елементи

    • Block element ,Например: div ; p
    • Inline element ,Например: image , a
    • Inline Block element

    HTML Tagove :

    • Headings: < h1 > to < h6 >
    • Paragraphs: < p > String < /p >
    • Absolute hyperlink: < a href="LINK ili ime na fail">ИМЕ< /a >
    • Как да отворим Hiper link в нов прозорец : target="_blank Пример: < a href="LINK ili ime na fail"target="_blank">Name< /a >
    • Images from internet : < img src > : < img src = "link"/>
    • Images from PC : < img src = "име на файла/име на снимката.png"/>
    • алтернатива ако снимката не се покаже да се покаже име : < img src = "link" alt="Name"/>
    • Ordered Lists : < ol > < /ol > списък с номерация 1.*** 2.*** 3.***...
    • Unordered Lists : < ul > < /ul > списък без номерация.
    • List : < li > < /li > когато имаме подточки в списък

  2. CSS

    3. CSS defines styling of the HTML elements

    Specifies fonts, colors, margins, sizes, positioning, floating, …

    CSS rule :

    • selector {prop1:val1;prop2:val2;..} ,Например : h1 {Font - size : 45px; color: yellow;}

    • External Style:

      < html >

      < head >

      < link rel="stylesheet" href="mystyle.css" >

      < /head >

      < body >

      < h1 >This is a heading< /h1 >

      < p >This is a paragraph.< /p >

      < /body >

      < /html >

    • rastoqnie v paragrafite : p {

      padding-bottom: 15px

      }

    • Можем да правим промени в определни заглавия,параграфи ,чрез : div class

      < div class="site-content/ или друго име"> ... в папката на CSS пишем точка после името и какво искаме да променим.< /div>
